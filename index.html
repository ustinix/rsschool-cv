Code example

Task from codewars: There was a test in your class and you passed it. Congratulations! But you are an ambitious person. You want to know if you are better than the average student in your class. You receive an array with your peers test scores. Now calculate the average and compare your score! Return true if you are better, else false!

Note: Your points are not included in the array of your class points. Do not forget them when calculating the average score!

Solution:

                            
                                function betterThanAverage(classPoints, yourPoints) {
                                    let total = 0;
                                    let avg = 0;
                                    for (i = 0; i < classPoints.length; i++) {
                                        total += classPoints[i];
                                        avg = total / classPoints.length;
                                    }
                                
                                    if (avg < yourPoints) {
                                        return true;
                                    } else {
                                        return false;
                                    }
                                }
